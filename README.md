# Projeto de Extensão - IFSP Campinas

Este repositório foi criado para centralizar o desenvolvimento e a organização do nosso projeto de extensão. Ele servirá como base tanto para a parte burocrática quanto para a parte técnica.

##  Sobre o Repositório
O projeto está em fase inicial e este espaço será utilizado para:

1. **Gestão de Projetos:** Armazenar todos os documentos e entregas da matéria de Gestão (cronogramas, escopo e planejamentos).
2. **Desenvolvimento:** Hospedar o código-fonte desenvolvido em **C**.
3. **Histórico:** Registrar a evolução das atividades de extensão exigidas pelo IFSP.
